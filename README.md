# 📱 SillyTavern Phone Extension

> ระบบโทรศัพท์จำลองแบบเต็มรูปแบบสำหรับ SillyTavern — เล่นโซเชียลมีเดีย แชท โทร และอีกมากมาย ภายในบอทตัวละครของคุณ!

---

## ✨ ฟีเจอร์ทั้งหมด

### 📱 ไอคอนลอยกลม (FAB)
- กดไอคอนวงกลมที่มุมขวาล่างเพื่อเปิดโทรศัพท์
- มีแถบ Dock ให้เลือก 5 หมวดหมู่

---

### 📸 Instagram
- **ดูสตอรี่** — เห็นสตอรี่ของ `{{Char}}` และ NPC อื่นๆ (ตัวละครเสริมเพิ่มเองอัตโนมัติ)
- **ฟีดโพสต์** — ส่องโพสต์รูป กด Like, Comment, Share ได้
- **DM (ข้อความส่วนตัว)** — แชทกับ `{{Char}}` โดย AI จะตอบกลับอัตโนมัติ
- **โปรไฟล์** — ดูโปรไฟล์ของคุณได้

---

### 💬 LINE
- **แชท** — ส่งข้อความ AI ตอบกลับ
- **โทร** — กดโทรหาตัวละคร (ทั้งเสียงและวิดีโอคอล มีหน้าจอโทรจริง + จับเวลา)
- **แชร์พิกัด** — กดส่ง Location ได้

---

### 🎵 TikTok
- **ฟีดวิดีโอ** — ดูวิดีโอสั้นของ `{{Char}}` และ NPC
- **กด Like / Comment / Share** ได้ทุกอย่าง
- **สไวป์ข้ามวิดีโอ** ด้วยปุ่ม Next

---

### 🐦 Twitter / X
- **โพสต์ทวีต** — เลือกสาธารณะหรือส่วนตัว
- **รีทวีต / Like / Reply** ได้หมด
- **ดูรายละเอียดทวีต** และคอมเม้นต์

---

### ⚙️ ตั้งค่า
- เปลี่ยน **วอลเปเปอร์โทรศัพท์** (6 แบบ)
- ดู **โปรไฟล์ผู้เล่น** (ชื่อ, เบอร์, ยอดเงิน)
- ดูวันเวลาปัจจุบัน

---

### 📝 บันทึก (Notes)
- เพิ่มบันทึกได้ไม่จำกัด
- **ตัวละครจะจดจำ** ข้อมูลที่คุณเขียนผ่าน AI prompt
- แก้ไข / ลบได้

---

### 🎵 เพลง
- **วางลิงก์ YouTube** เพิ่มเพลงได้หลายเพลง
- **เล่น / หยุด / ข้าม / ย้อนกลับ** ได้หมด
- **ปรับระดับเสียง** ด้วย Slider
- ดูวิดีโอ YouTube แบบ Embed ในแอพ

---

### 🌤️ สภาพอากาศ
- แสดงสภาพอากาศ **ใน World ของตัวละคร**
- อุณหภูมิ, ความชื้น, ลม, ทัศนวิสัย
- เวลาพระอาทิตย์ขึ้น-ตก

---

### 🐾 เกมเลี้ยงสัตว์
- เลือกสัตว์เลี้ยง 6 ประเภท (แมว, หมา, กระต่าย, แฮมสเตอร์, จิ้งจอก, หมี)
- **ให้อาหาร, อาบน้ำ, แปรงขน, เล่นด้วย, พาเดิน, แกล้ง**
- มีบาร์ความหิว / ความสุข / ความสะอาด
- **ไม่กระทบการ Roleplay** — เป็นระบบแยกสำหรับเล่นระหว่างรอ AI ตอบ

---

## 📦 การติดตั้ง

### วิธีที่ 1: ผ่าน Extension Manager (แนะนำ)
1. เปิด SillyTavern → ไปที่ **Extensions** → **Install Extension**
2. ใส่ URL: `https://github.com/YOUR_USERNAME/sillytavern-phone-extension`
3. กด Install และ Reload

### วิธีที่ 2: Manual
1. Clone repo นี้ลงใน folder:
   ```
   SillyTavern/public/scripts/extensions/third-party/sillytavern-phone-extension
   ```
2. Reload SillyTavern
3. ไปที่ Extensions → เปิดใช้งาน **Phone System**

---

## 🎮 วิธีใช้งาน

1. **เลือกตัวละคร** ที่ต้องการเล่น
2. **เริ่ม Chat** กับบอทนั้น
3. **กดไอคอนวงกลม** 📱 ที่มุมขวาล่าง
4. เลือกแถบที่ต้องการจาก Dock
5. กดเข้าแอพ และสนุกได้เลย!

> 💡 **Tip**: ตัวละครหลัก (`{{Char}}`) จะถูกเพิ่มในทุกแอพโดยอัตโนมัติ!

---

## ⚙️ Requirements

- SillyTavern เวอร์ชัน **1.10.0+**
- Extension ต้องมีสิทธิ์ใช้ `generateQuietPrompt` (เปิดอยู่โดย default)
- สำหรับเพลง: ต้องการ URL YouTube ที่ถูกต้อง
